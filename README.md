# achivements

## このリポジトリについて

このリポジトリは私の業績を管理するためのリポジトリです。業績一覧は書類によって求められる情報が異なるため、大変です。researchmapを使えばいくらかラクにはなりますが、必要とされる情報のすべてをresearchmapがフォローしているわけではありません。そこで、必要に応じて適宜情報を加えられるように、YAML形式でデータを記録し、これをRスクリプトで形式変換する仕組みを作成しました。

## 業績データの入力方法

このプロジェクトでは、1つの業績を1つのYAMLファイルとして管理します。データを新規に作成する場合、新しくYAMLファイルを作成し、必要な項目を記述します。このとき、VSCodeのスニペット機能を使うことができるように設定されているため、「pub」と入力すれば、業績（論文，発表等）のテンプレートが挿入されます。あとはこれに必要な情報を記入していくだけです。

ファイル名に制約はありませんが、私は管理しやすいように「yyyy-MM-keywords.yaml」という形式で命名しています。yyyyは発表年、MMは発表月、keywordsは業績を特定するためのキーワードです。

## ディレクトリ構成

ディレクトリ構成は以下の通りです。

```txt
achievements/
├─ .vscode/
│  └─ achievements.code-snippets # 業績用YAMLテンプレート
│
├─ data/
│  ├─ 2026-01-keywords.yaml # 2026年1月の業績
│  ├─ 2025-07-keywords.yaml
│  └─ ...
│
├─ scripts/
│  └─ format-conversion.R # 業績データのフォーマット変換スクリプト
│
└─ README.md # このファイル
```
